# YouTube-Video-Installer-Mp3/Mp4
A robust Python CLI tool to download YouTube videos (MP4) or extract audio (MP3). Built with pytubefix, it features auto-folder organization, continuous downloading loops, and automatic audio file renaming. It handles errors gracefully and saves files directly to your local documents, making media archiving fast and easy.

TO RUN THE CODE YOU NEED:

python -m venv myenv
.venv/Scripts/Activate.ps1 
pip install pytubefix (or requirements.py)
